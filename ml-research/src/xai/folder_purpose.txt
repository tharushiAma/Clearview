Purpose: Explainable AI (XAI) suite for model interpretability.

Files:
1. Explainable.py (11.3 KB) - CORE XAI ENGINE
   - ClearViewExplainer class with multiple explanation methods:
     * explain_ig_aspect() - Integrated Gradients for aspect sentiment
     * explain_ig_conflict() - Integrated Gradients for conflict detection
     * explain_lime_aspect() - LIME explanations
     * explain_shap_aspect() - SHAP value calculations
     * explain_msr_delta() - MSR before/after comparison (unique contribution)
   - GPU-accelerated attribution computations
   - Token-level attribution scores

2. run_xai.py (2.1 KB) - CLI WRAPPER
   - Command-line interface for running XAI explanations
   - Bundles all explanation methods into a single report
   - Example: python run_xai.py --ckpt model.pt --text "review" --aspect price

Used to prove MSR effectiveness by showing which tokens drive predictions and how MSR changes them.
Generated the GPU verification reports in outputs/xai/.
