Purpose: Model evaluation and metrics calculation.

Files:
1. evaluate_and_log.py (25 KB) - MAIN EVALUATION SCRIPT
   - Comprehensive evaluation suite with CLI interface
   - Calculates ABSA metrics (precision, recall, F1 per aspect)
   - Conflict detection metrics (AUC, separation, Brier score)
   - MSR error reduction analysis
   - Saves results as JSON and CSV
   - Used to generate all thesis tables

2. collate_thesis_results.py (2.1 KB)
   - Aggregates results from multiple ablation runs
   - Creates the 8-config comparison table (total_results.md)

3. compare_reports.py (3.2 KB)
   - Compares two evaluation reports (baseline vs MSR)
   - Statistical significance testing

4. make_thesis_tables.py (2.5 KB)
   - Formats evaluation results into publication-ready CSV tables
   - Generates thesis_table_main.csv, thesis_table_per_aspect.csv, etc.

Used to generate the CSV tables in outputs/eval/ for your thesis results.
