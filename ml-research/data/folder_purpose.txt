Purpose: Contains all data splits and augmented datasets for the 4-class ABSA+MSR system.

Structure:
- raw/ - Original unprocessed data
- processed/ - Cleaned and standardized datasets
- splits/ - Train/validation/test splits (train.parquet, val.parquet, etc.)
- augmented/ - Augmented datasets (synthetic, sampler-based)

Key Files:
- splits/train.parquet - Original training set
- splits/train_aug.parquet - Fully augmented training set (Original + Sampler + Synthetic)
- splits/val.parquet - Validation set
- class_counts.json - Class distribution statistics
